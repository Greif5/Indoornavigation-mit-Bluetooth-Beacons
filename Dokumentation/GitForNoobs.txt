Download-Link für Git for Windows:
https://git-for-windows.github.io/
Version: 2.14.2

Jede aktion wird mit dem Befehl Git ausgeführt.
Z.B.: git status
	Zeigt den status des aktuellen Repositories an.
	Z.B. ob sich Dateien geändert haben, ob sich das Repository geändert hat, etc.
	
git clone https://github.com/Greif5/Indoornavigation-mit-Bluetooth-Beacons
	klont das Repository in das aktuelle Verzeichnis der Konsole.
	Z.B. D:\...
	
git add
	merkt eine Datei für einen Commit an.
	Das muss jedesmal gemacht werden wenn ein Commit gemacht werden soll.

git commit
	erstellt einen commit.
	Zu einem commit muss auch immer ein Test eingetragen werden.
	
git push
	pusht die aktuell vorgemerkten commits auf das repository.
	
git rm
	löscht ein Verzeichnis / Datei aus dem Gedächtsnis von Git.
	Es trackt keine Veränderungen an der Datei mehr.
