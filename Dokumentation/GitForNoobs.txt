Einfach runterladen und beim Installieren alle Einstellung auf standard lassen.



Befehle:

"git help" 
	zeit das manual
	
"git help <command>" 
	öffnet im interent ein help page zum kommmando

"git clone https://github.com/Greif5/Indoornavigation-mit-Bluetooth-Beacons"
	Kopiert das komplette reposotorie in den Ordner wo der Befehl ausfgeüfhrt wird. (mit allen branches ect.)
	
	
"git status"
	- zeigt an auf welchem branch man gerade arbeitet.
	- zeigt an welche änderungen man auf dem zum akutllen branch gemacht hat.
	- zeigt die nächten auszuführenden befehele an. 
	
	
"git add hallo/test"
	Fügt die datei dem aktuellen abbild hinzu (abbild was man später pushen will.

	
"git commit"
	Update das file im Reposotory, wenn man git lokal benutz wird allerdings noch "git push" benutzt.
	Es werden alle files geupdated die vorher mit "add", "rm" ect. hinzugefügt worden.

"git push" 
	läd alle commit ins online repositry (github).

	
"git config core.editor notepad++"
	öffnet notepad für text eingabe z.B. für commit
